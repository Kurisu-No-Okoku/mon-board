📢 Notes de version

v1.10.1 - Correctifs UI double tableau

Publié le 3 Mai 2026

  • Fix (UI) : La colonne "Liste de mots" ne dépasse plus la hauteur du tableau principal — les deux colonnes s'alignent dynamiquement via flexbox.
  • Fix (UI) : Bouton "Nouvelle entrée" ➕ avec un fond légèrement vert pour le distinguer du bouton Import/Export en mode sombre.

v1.10.0 - Import/Export & Nouvelle entrée

Publié le 3 Mai 2026

  • Fonctionnalité : Bouton ↕ Import/Export en haut à droite du titre Orthophoniste — import CSV et export direct en fichier CSV.
  • Fonctionnalité (Admin) : Bouton ➕ Nouvelle entrée visible uniquement pour l'admin — saisie manuelle avec recherche ou création automatique du mot dans la table Mots.
  • Fix (UI) : Message "En construction" centré correctement dans les sections Sessad, ULIS et Maison.

v1.9.2 - Hotfix navigation bouton "Liste de mots"

Publié le 3 Mai 2026

  • Fix (critique) : Le bouton "Liste de mots" partage la classe category-button, ce qui le faisait capturer par le listener de navigation et appelait setCategory(undefined) — fermant le tableau et bloquant le chargement. Ajout d'un guard if (!data-category) return.

v1.9.1 - Hotfix double tableau

Publié le 3 Mai 2026

  • Fix : Colonne "Mot(s)" affichait "Plusieurs" quand Total vaut NULL en base — désormais correctement affiché comme "Aucun".
  • Fix : Bouton "Liste de mots" ne déclenchait pas le chargement — comparaison d'état du display corrigée (!== 'none' → === 'block') et valeur d'affichage explicite.

v1.9.0 - Vue double tableau Orthophoniste

Publié le 3 Mai 2026

  • Fonctionnalité : Nouveau bouton "Liste de mots" qui apparaît après le chargement du tableau orthophoniste.
  • UI : Affichage en deux colonnes — liste de mots à gauche, tableau de suivi à droite (empilement vertical sur mobile).
  • Navigation : Clic sur un mot filtre et surligne les lignes correspondantes dans le tableau ; clic sur une ligne surligne le mot dans la liste.
  • Fix : La colonne "Mot(s)" affiche désormais "Aucun" (sans italique) quand le total est à 0.

v1.8.0 - Sécurité & Authentification

Publié le 3 Mai 2026

  • Sécurité (Auth) : Mise en place d'un système de tokens de session côté serveur — chaque connexion génère un token unique invalidé à la déconnexion.
  • Sécurité (Endpoints) : Protection des routes API par rôle : les endpoints admin exigent désormais un rôle Admin, les autres un utilisateur connecté.
  • Sécurité (XSS) : Échappement HTML systématique dans la modale de gestion des utilisateurs et dans les réponses serveur.
  • Login : Seul le compte Admin peut se connecter sans mot de passe — les autres comptes en attente reçoivent un message d'erreur explicite.
  • Fix (Runtime) : Correction de la variable host non définie dans la config SQL Server (ReferenceError au démarrage).
  • Fix (UI) : Correction des éléments manquants dans le DOM (construction-message).
  • UI : Correction du hover du bouton Accueil en thème clair, fusion du CSS dupliqué, ajout de l'animation fadeIn.

v1.7.5 - Navigation & UI Fixes

Publié le 3 Mai 2026

  • Fix (Navigation) : Correction de la logique d'affichage des catégories pour éviter les retours inattendus à l'accueil.
  • Fix (UI) : Amélioration du style et du centrage du bouton de réinitialisation de mot de passe dans la modale "Mon Compte".
  • Synchronisation : Mise à jour de la version du site et de l'API à v1.7.5.

v1.7.4 - Accès Extérieur & Cloud Ready

  • Support des accès via domaines externes et détection dynamique de l'URL de l'API.

v1.7.3 - Orthophoniste & Import Fix

  • Finalisation de l'affichage dynamique des mots et du suivi orthophoniste.
  • Nouveau parser CSV ultra-robuste capable de détecter la date n'importe où dans la ligne.
  • Affichage complet des informations de compte incluant l'adresse e-mail.

v1.7.0 - Parser CSV Auto-adaptatif

Publié le 3 Mai 2026

  • Fix (Parser) : Détection dynamique de la colonne date pour supporter les exports Numbers/Excel avec colonnes vides.
  • Synchronisation : Harmonisation de la version v1.7.0 sur l'ensemble du projet (Site, API, Package).
  • Stabilité : Amélioration du typage des données lors de l'insertion SQL (sécurité sur les valeurs NULL).

v1.6.9 - Stabilité Import

v1.6.8 - Import CSV Orthophoniste

  • Parsing : Support des mots uniques et des listes de mots pour le suivi.

v1.6.6 - Stabilité & Logs

v1.6.4 - Import CSV & Tables

  • Navigation : Redirection systématique vers l'accueil après chaque connexion ou rechargement.
